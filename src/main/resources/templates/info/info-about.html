About Unect
Unect는 University + Connect를 합한 합성어 입니다.
모든 학우들이 편안한 서비스를 이용할 수 있도록 노력하겠습니다.
Page Programming
저희 페이지는 JSP를 이용하여 만든 페이지가 아닌 JAVA + SPRING BOOT로 개발된 페이지입니다.
학교에서 배우는 JSP에 대한 한계가 있어 졸업작품을 시작하기 전 2달간 서로 공부하여 개발하였습니다.
