About Unect
모든 학우들이 편안한 서비스를 이용할 수 있도록 노력하겠습니다.
팀원 소개
안녕하세요 저희는 남서울대학교 멀티미디어학과 가준영, 이지훈입니다.
저희 프로젝트는 Github에서 확인하실 수 있습니다.
페이지 소개
Unect는 University + Connect의 합성어 입니다.
현재는 저희 학과에 머물러 있지만 나아가 모든 대학의 각 학과 커뮤니티로 발전시킬 예정입니다.
페이지 개발 배경
Unect Project는 JAVA + SPRING BOOT + Thymeleaf로 개발된 페이지입니다.
학교에서 배우는 JSP에서 더 나아가보자 졸업작품을 시작하기 전 2달간 서로 공부하여 개발하였습니다.
